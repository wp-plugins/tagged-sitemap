Creates a shortcode [tagged-sitemap] that renders a nested list with all the pages and tags for page. When a tag is clicked, related pages are then highlighted.

== Installation ==

1. Download the plugin package 'tagged-sitemap.zip'
2. Unzip the package and upload to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
